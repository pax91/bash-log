### Description
Import as source the log.sh file into your script.
Call the log function with the following params accepted (uppercase or lowercase) no order required.
```bash
log cyan bold "Hello World"
```
### Allowed Colors
- black
- red
- green
- yellow
- blue
- purple
- cyan
- white
### Allowed Modes
- regular
- bold
- lowintensity
- underline
- blinking
- reverse
- invisible
